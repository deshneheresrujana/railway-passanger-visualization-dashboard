Railway Passenger Visualization Dashboard
Overview

The Railway Passenger Visualization Dashboard is a data analytics project designed to analyze and visualize railway passenger information using interactive charts and dashboards. The project focuses on transforming raw passenger data into meaningful insights to help understand travel patterns, demographics, and ticket distribution.

This dashboard allows users to explore passenger data such as source station, destination station, passenger ID, age, gender, and ticket details through clear and interactive visualizations.

Objectives

To organize railway passenger data in a structured format.

To analyze passenger travel patterns between different stations.

To create visual dashboards for better understanding of passenger demographics.

To demonstrate data analysis and visualization skills using business intelligence tools.

Features

Interactive dashboard for passenger data analysis

Visualization of source and destination travel patterns

Passenger demographic insights (age and gender distribution)

Ticket-based passenger tracking using Passenger ID

Multiple charts for easy interpretation of railway data

Dataset Information

The dataset used in this project contains passenger travel details including:

Passenger Ticket ID

Passenger Name

Age

Gender

Source Station

Destination Station

The dataset was cleaned and structured before being used for visualization.

Tools and Technologies Used

Power BI / Tableau – Dashboard creation and visualization

Microsoft Excel – Data cleaning and preparation

SQL (optional) – Data querying and structuring

Dashboard Visualizations

The dashboard includes the following visual elements:

Passenger distribution by source and destination

Gender-wise passenger analysis

Age group distribution

Travel frequency between stations

Passenger ticket identification insights

These visualizations help in identifying patterns and trends in railway passenger travel.
